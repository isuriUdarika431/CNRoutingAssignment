Description
This repository contains the Cisco Packet Tracer files developed for the Computer Networks Practical Assignment.

The assignment includes:

Basic Router Configuration
Static Routing Configuration
RIP Dynamic Routing
EIGRP Dynamic Routing

All configurations were successfully implemented and tested using Cisco Packet Tracer. Communication between all networks was verified using routing tables and successful ping tests.

Files Included

1. Task 1.pkt

Contains:

Basic router configuration
Hostname configuration
Console password configuration
Console login authentication

2. Task 2.pkt

Contains:

IP addressing configuration
Static routing configuration
Successful communication between Network A, Network B, and Network C

3. Task 3_RIP.pkt

Contains:

RIP Version 2 configuration
Dynamic route exchange between routers
Successful communication between all networks using RIP

4. Task 3_EIGRP.pkt

Contains:

RIP removed
EIGRP configured
Successful communication between all networks using EIGRP
